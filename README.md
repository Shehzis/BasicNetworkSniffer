# Basic Network Sniffer
CodeAlpha Internship - Task 1

## Description
A Python-based network packet sniffer built using Scapy.
Captures and analyzes live network traffic, displaying source/destination IPs,
protocols (TCP/UDP/ICMP), ports, DNS queries, and payloads.

## Features
- Real-time packet capture
- TCP, UDP, ICMP protocol analysis
- DNS query and response parsing
- Payload inspection (text and hex)
- Protocol summary and log file export

## Requirements
- Python 3.x
- Scapy: `pip install scapy`
- Windows: Npcap (https://npcap.com)
- Run as Administrator (Windows) or sudo (Linux/Mac)

## Usage
python network_sniffer.py

## Tools Used
- Python, Scapy, Npcap
